Unique Features

What make us different? Here some cool, unique features of BluPoint SmartTV Framework

Open Source

Yes it's completely an open source project. We started to develop and will grow with your commits.

Native Javascript

No, we are not joking. There is no external libraries such as jQuery.
Just EcmaScript 6

Modularity

As we discussed to write this framework in ES6. We tried to use its all features like classes, modules etc. As result we prepared readable, clean code

Cross Platform

It supports most of SmartTV manufacturers. We are supporting old Opera (Presto Engine) too.

DRM-Ready

DRM's are indispensable part of VOD content providers. Our framework includes its own DRM implementations for SmartTV's with their own DRM Engine

Multi Audio

You don't need to confuse with audio change for your digital content. We care that!

VMAP & VAST ADS Supported

Fully integrated VMAP parsing with IAB's Vmap standard. Vast render and seamless integration

Tons of features

We are offering a lot of features like Subtitle, Event, Storage etc. You can find them in documentation.

How to initialize

Our main focus is speed, ease of use and newest technology. Before reading documentation we wanted to show how to initalize BluPoint SmartTV Framework in your project.

copy 


// We assumed you included library to your project
//from npm, yarn, bower or directly.

let Device = new Device();
let Storage = new Storage();

console.log(Device); // Thats all, you can use modules like Events, Player etc :)
console.log(Storage);
